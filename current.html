Modeled Migration Routes Web and Outreach Maps

Data were gathered from the massive GPS collar database using SQL to pull out specific data. A Brownian Bridge Migration Model was ran on the data to find areas of high and low use for migration. A 3D map using ArcGIS JavaScript API is being created to allow users to better visualize the routes and understand them through multiple popup info graphs. The purpose of the project is to create visually appealing maps and easy to use web maps to allow stakeholders and the public to understand the purpose of the migration initiative.
