ArcGIS JavaScript API Web Map

The purpose of the project was to create a web map utilizing ArcGIS JavaScript API. Using freshwater public supply water use, the total consumption per person by county was calculated. Three services were published and referenced in the JavaScript code. A visual re-render was used on the total water use per person service to display it with circles based on the average total gallons used per person per day. Multiple other widgets and features such as popups, a legend, a search function, and title were added.
